# YouTube Clickbait Detector

CLI-утилита для выявления кликбейтных видео по метрикам CTR и удержания.

## Требования
- Python 3.8+
- Установка зависимостей: `pip install tabulate pytest`

## Использование
```bash
python main.py --files data1.csv data2.csv --report clickbait
